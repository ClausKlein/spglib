spglib
======

|Version Badge| |Downloads Badge| |Binstar Badge| |Build Status|

C library for finding and handling crystal symmetries

How to compile
--------------

::

    % aclocal
    % autoheader
    % libtoolize # or glibtoolize with macport etc
    % touch INSTALL NEWS README AUTHORS
    % automake -acf
    % autoconf
    % ./configure
